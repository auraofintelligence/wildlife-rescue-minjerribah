# Offline map assets

`minjerribah.pmtiles` is an extract of the Protomaps daily basemap dated
19 July 2026, bounded to Minjerribah and nearby waters at zoom levels 0–15.

- Basemap source: <https://build.protomaps.com/20260719.pmtiles>
- Map data: OpenStreetMap contributors, Open Database Licence
- Basemap software and styles: Protomaps, BSD-3-Clause / CC0 as applicable
- Renderer: MapLibre GL JS

The app must retain visible OpenStreetMap attribution whenever the map is shown.

The bundled font glyphs and sprites come from the
[`protomaps/basemaps-assets`](https://github.com/protomaps/basemaps-assets)
repository. Their upstream licence notices apply.
